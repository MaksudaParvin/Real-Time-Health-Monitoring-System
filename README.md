Developed an IoT-based healthcare monitoring system capable of tracking patients' vital signs in real time, including body temperature, heart rate (BPM), blood oxygen saturation (SpO₂), and motion detection. The system integrates multiple sensors with an ESP32 microcontroller to collect and process health data continuously. 
Integrated an OLED display for live data visualization and the Blynk platform for remote monitoring. Implemented automated alert and email notification features to instantly report abnormal health conditions, improving patient safety and emergency response.
